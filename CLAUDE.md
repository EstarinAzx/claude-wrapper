# claude-wrapper

Electron app wrapping the Claude Code CLI — Claude Code runs under the hood, a web UI replaces the raw terminal.

## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues on `EstarinAzx/claude-wrapper` (via `gh`); external PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical five-label vocabulary, unmodified (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` + `docs/adr/` at repo root (created lazily). See `docs/agents/domain.md`.
